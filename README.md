# Digital Circuit Lab (DCLab)

This repo includes all of the HWs of Digital Circuit Lab (DCLab).

## Requirements

- Quartus 16
- DE2-115 FPGA board

## Overview

- Lab1: Random number generator
- Lab2: RSA decoder
- Lab3: Audio recorder
